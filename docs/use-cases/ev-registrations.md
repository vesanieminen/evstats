# Use Case: Monthly EV Registrations

## Goal
Visualize monthly BEV registration counts.

## Actors
- Analyst or enthusiast

## Preconditions
- Application is running.
- EV registration dataset is available in app resources.

## Steps
1. Open the New cars view from the Statistics navigation group.
2. Inspect the monthly column chart.
3. Use timeline controls to zoom the period of interest.

## Result
- A timeline column chart shows BEV registration counts per month.

## Data Sources and Dependencies
- CSV data loaded from application resources (EV registrations by month).
- Vaadin Charts for the interactive timeline.
