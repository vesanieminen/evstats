# Use Case: Tesla Registrations by Year and Month

## Goal
Compare Tesla registrations by month across multiple years in a stacked bar chart.

## Actors
- Analyst or enthusiast

## Preconditions
- Application is running.
- Tesla registration dataset is available in app resources.

## Steps
1. Open the New Teslas / year view from the Statistics navigation group.
2. Review the stacked bar chart with months as series and years as categories.

## Result
- A stacked bar chart compares Tesla registrations per month across years.

## Data Sources and Dependencies
- CSV data loaded from application resources (Tesla registrations by month).
- Vaadin Charts for stacked bar rendering.

## Notes
- When a Vaadin Charts license is not configured, the view displays a license notice instead of the chart.
