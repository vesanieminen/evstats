# Use Case: Tesla Monthly Registrations

## Goal
Track Tesla registration counts over time.

## Actors
- Analyst or enthusiast

## Preconditions
- Application is running.
- Tesla registration dataset is available in app resources.

## Steps
1. Open the Tesla registrations view from the Statistics navigation group.
2. Inspect the monthly column chart.
3. Use timeline controls to zoom the period of interest.

## Result
- A timeline column chart shows Tesla registrations per month.

## Data Sources and Dependencies
- CSV data loaded from application resources (Tesla registrations by month).
- Vaadin Charts for the interactive timeline.
